3. 痛點、爽點、癢點都是好的產品切入點，但必須熟悉目標受眾才能精準切入

使用 comparison 結構

## Joey 的寫作風格特徵

### 1. 結構與節奏

**開場方式：**
- 直接提問法：「怎麼培養『遇到貴人』的體質？」、「為什麼短影音做不起來？」
- 故事開場法：「幾個月前去農訓協會...」、「在經營 ShiFu 之前，我是做電商的...」
- 觀點陳述法：「我一直很相信，『競爭心態，會傷害競爭力。』」
- 引發對話法：「最近很多人私訊問我...」

**段落組織：**
- 優先使用編號列表（1、2、3）
- 每條信息獨立成句，避免過長段落
- 使用箭頭符號表示因果關係：「→」
- 使用冒號清晰標示要點與解釋

### 2. 用字遣詞特色

**口語化但不失專業：**
- 「貴人閉環、貴人永動機」（創新複合詞彙）
- 「一頓操作猛如虎，一看業績5萬5」（網路梗）
- 「屎在肚中滾，找不到廁所」（極其白話的類比）
- 「尿點」（業內專用術語）

**對比與反轉：**
- 「不是贏，而是先不要死」
- 「不是未被滿足的需求，而是背後有恐懼和痛苦的需求」
- 「傾聽不是被動，是掌控全局的主動」
- 「越深刻的道理，往往越簡單」

**比喻與類比：**
- 「信用額度」（借用金融概念）
- 「一魚多吃」（簡潔有力的比喻）
- 「賽馬不猜馬」（經典電商術語）
- 「虎頭豹尾」（古文成語）

### 3. 語氣與情感

**親近而有權威：**
- 經常使用第一人稱分享實戰經驗
- 「我在他身上學到的 4 個...」
- 「我一直很相信...」
- 「我的短影音課...」

**謙虛但自信：**
- 分享失敗案例：「半年拍了 40 支影片，平均觀看只有 500」
- 但隨即展示成功：「後來我把方法套回自己身上，影片觀看數變 40 幾萬」
- 「回頭來看，曾經投入過的努力，都不算白費」

**鼓勵與實用導向：**
- 「如果想要遇到貴人，你自己也必須是貴人」
- 「敢開口、敢尋求協助，就有機會」
- 「讓數據告訴你答案」

### 4. 數字的使用

**具體化抽象概念：**
- 「100萬」、「250%」、「400倍」
- 「700萬」觀看、「2.2萬」粉絲、「1.1萬」人轉換
- 「150萬」直播業績、「兩億」貸款業績
- 「年營收四億」、「公司估值五億」

**突出成長與對比：**
- 「從 0 漲到 2.2 萬」
- 「從零做到四萬粉絲」
- 「四個月從零漲到 3.7 萬」
- 「觀看數從 400 變 40 幾萬」

### 5. 標點符號使用

**引號的多層次運用：**
- 單引號表示術語/概念：『信用額度』、『傾聽』、『第三方案』
- 雙引號表示引用/強調：「機會」、「自己成長」
- 中文句號 + 破折號做強調連接

**破折號的節奏作用：**
- 「痛點是『撫平客戶恐懼的需求』——不是未被滿足的需求，而是背後有恐懼和痛苦的需求」
- 破折號用於解釋說明，製造轉折與拆解

**括號補充：**
- 提供上下文或數據支撐
- 「線上課程變現課（線上課程、拍攝剪輯實體課、團體輔導、線上諮詢四大區塊）」

### 6. 常見修辭手法

**排比：**
- 「過去做電商抓到 Facebook 廣告紅利和直播紅利，三年營收近 4 億。現在經營 ShiFu 抓到短影音紅利」
- 「賺錢（利潤全留）、以教為學（理解提升250%）、擴大影響力匯聚人才、測試素人開課的商業模式」

**遞進：**
- 「獲得幫忙 → 做出成績 → 反過來幫人 → 獲得更多貴人」
- 「獲得幫助 → 獲得更多幫助」的循環論證

**對偶：**
- 「當你把重心放在對手，就看不到遠方；當重心放在『自己成長』，就能關注對手」

### 7. 內容類型與主題

**經常涉及的領域：**
- 短影音操盤（最頻繁）
- 線上課程運營
- 創業心態與策略
- 商業變現
- 人脈與談判
- 電商與團隊

**故事與案例導向：**
- 「農會大姐」案例
- 「Sandy 帳號」案例
- 自身電商經驗
- 與王金平的合作故事

### 8. 行動導向

**呼籲性強：**
- 「歡迎你分享給想創業的朋友。」
- 「我的短影音變現課開賣了」
- 「說明會今年最後加開」

**轉化意圖明確：**
- 分享實戰案例 → 建立信任 → 推薦課程
- 指出問題 → 提供解決方案 → 購買課程

## 寫作節奏與文字數量

- **Hook（開場）**：短句，引發好奇（20-40 字）
- **Core Message（核心）**：3-4 個要點，每個 15-30 字
- **Supporting Quote（佐證）**：精煉金句（10-25 字）
- **Total Per Post**：300-500 字（分解成多條信號）

## 推薦應用場景

✅ 當需要「教學型貼文」時，使用 Joey 的：
- Hook + 3-4 個 Key Message + 金句引用的套路
- 「編號列表」的組織方式
- 「反轉式」的論點表達

✅ 當需要「故事分享」時，使用 Joey 的：
- 「Before/After」結構
- 「從失敗到成功」的敘述弧線
- 具體數字支撐

✅ 當需要「行業洞見」時，使用 Joey 的：
- 「對比論證」
- 「遞進邏輯」
- 「第三方案」的思維框架
